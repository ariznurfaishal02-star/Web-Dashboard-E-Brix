# Web-Dashboard-E-Brix
E-BRIX adalah platform monitoring interaktif berbasis web yang dirancang untuk memvisualisasikan tingkat kemanisan (Brix) tebu secara spasial. Proyek ini mengintegrasikan teknologi Cloud Computing, Geospatial Analysis, dan Web Development untuk membantu manajemen pabrik gula dalam mengoptimalkan jadwal panen
